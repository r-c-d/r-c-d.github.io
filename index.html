Read Data from Unshared Googlesheet

Googlesheet read from: https://docs.google.com/spreadsheets/d/1LzsCyRJl5ORv16i3dOVQr4VQBrSYnmYGmY6HvEF7hy8/edit#gid=0

Place this HTML file in a directory. Include the 'client_secret.json' file in the same directory.
Open a console and navigate to the directory. Run the command '$ python -m http.server 8000' from in the directory to simulate a server.
Open 'http://localhost:8000/' in a browser.
Click 'Sign in to fetch data' button.
